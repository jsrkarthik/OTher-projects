Current directory-HW1 should have ./IR directory(input data)
hw1.py contains solution for the first part of the assignment -Boolean retrieval
    Running steps
       a) Use command python hw1.py to run the program
       b) Enter the query when requested in the console
       c) A list of files that are matched by Boolean retrieval method will be produced
